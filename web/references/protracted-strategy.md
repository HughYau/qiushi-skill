# 持久战略

> "在战略上我们要藐视一切敌人，在战术上我们要重视一切敌人。也就是说在整体上我们一定要藐视它，在一个一个的具体问题上我们一定要重视它。"
> —— 毛泽东（1958年）

## 核心原则

**面对复杂长期任务，既不急于求成（速胜论），也不畏难放弃（悲观论），而是划分阶段、稳步推进。战略上保持信心，战术上每一步都严谨对待。**

## 不适用场景

- 任务可以在当前会话内完成
- 用户只需要一个具体步骤的执行结果
- 问题已经有清晰的解法，只缺执行
- 处于某一阶段的具体执行中，不需要重新评估战略

## 方法流程

### 第一步：客观评估态势

**我方条件：** 当前拥有的资源、能力、信息；已取得的成果；存在的弱点
**任务条件：** 总体规模和难度；可预见的困难；有利的条件和机遇

**关键：** 既不夸大困难（导致悲观），也不低估困难（导致冒进）。实事求是。

### 第二步：拒绝速胜论和悲观论

- **速胜论：** "这很简单，马上就能搞定"
- **悲观论：** "这太复杂了，做不到"
- **正确态度：** "这需要持久的努力，但最终可以完成。关键是分阶段推进。"

### 第三步：划分阶段

**第一阶段·战略防御（探索与积累）：**
- 目标：站稳脚跟，保存实力，积累认识
- 方法：调查研究，建立基础认知
- 心态：不急于求成，允许"退让"以保存力量

**第二阶段·战略相持（最困难也最关键）：**
- 目标：逐步积累优势，等待转折点
- 方法：积小胜为大胜，在每个具体问题上集中兵力
- 心态：耐心，这是"最困难的时期，然而它是转变的枢纽"

**第三阶段·战略反攻（突破与收获）：**
- 目标：利用积累的优势，全面推进
- 方法：条件成熟时果断出击
- 心态：自信但不大意

### 第四步：在防御中求进攻

即使在整体处于困难阶段，也要在具体的、局部的问题上采取主动：
- 战略上是持久战，战术上求速决
- 用一个一个的小胜利积累信心和优势

### 第五步：识别转折点

持续观察条件变化，识别从相持转向反攻的时机。当条件成熟时，果断转入全面推进。

## 操作规程

当本方法被触发时，**输出一份阶段评估表和行动计划**：

1. **判断当前阶段**（必须明确选择一个）：
   - `战略防御期`：目标明确但能力/资源暂时不足
   - `战略相持期`：已有一定基础，正在积累优势
   - `战略反攻期`：条件成熟，可以全面推进

2. **填写阶段评估表**：
   ```
   当前阶段：[战略防御/相持/反攻]
   判断依据：……
   我方优势：……
   我方劣势：……
   关键障碍：……
   本阶段的核心任务（1条）：……
   本阶段的禁止事项：……
   转入下一阶段的条件：当 [具体条件] 时，转入 [下一阶段]
   ```

3. **在防御/相持期，必须同时给出"局部进攻点"**

4. **设置阶段检查点**：明确说明"在完成 [具体里程碑] 后，重新评估阶段"

## 附录：阶段评估指南

### 战略防御阶段的信号
- [ ] 对问题域的理解还很表面，经常发现新的未知
- [ ] 尝试的方案频繁失败或需要大幅调整
- [ ] 没有可靠的基础设施/工具链可以依赖
- [ ] 外部约束限制了行动空间

### 战略相持阶段的信号
- [ ] 已有初步的认知和基础，但离目标仍有明显距离
- [ ] 进展存在但缓慢，容易感到焦虑或疲惫
- [ ] 遇到了一些"硬骨头"问题尚未攻克
- [ ] 已积累的经验开始发挥作用，但尚不足以决定性突破

### 战略反攻阶段的信号
- [ ] 对问题的理解已经深入，方案清晰可靠
- [ ] 基础设施和工具已经到位，执行效率高
- [ ] 主要障碍已被消除或大幅削弱

### 阶段转化关键信号
- **防御→相持**：对问题有了系统性认知；有了可靠的根据地；能稳定解决局部问题
- **相持→反攻**：积累的小胜利量变引发质变；关键障碍被突破；执行信心达到临界点

**注意：** 阶段转化是渐进的，不要期待一个明确的"现在可以反攻了"的信号。
